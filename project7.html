To Do App
PRIMER PROYECTO EN JAVASCRIPT.

To Do App es mi primer proyecto hecho con javascript desde cero, a modo de app. Aprendí a agregarle a mi documento base nuevos elementos desde mi hoja de javascript y a crear dinamismo mediante la lógica, lo que hace interactivo mi proyecto. Cumple la funcionalidad de un ejecutador de tareas y permite organizarse en base a ellas. Este proyecto me permitió aprender a manipular el DOM y crear interactividad con el usuario.
